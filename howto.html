Creating Cards

1. Click either button pointed above.

2. You will see the edit page.

3. You can now enter the word for the card.

4. And also the description which the word needs to be matched with.

5. Press the plus button to add the card.

6. Your newly added card will appear in the table.

7. You can continue adding other cards.

8. If you decided to delete a card, just press the minus button that corresponds to the card.

9. Press OK to confirm the deletion.

Starting a Quiz

1. You must add some cards before you can start a quiz.

2. If you have already added your cards, press either button shown above to start a quiz.

3. The quiz has been started if you see something similar to what is shown above.

4. Click the white square cards to turn open it.

Giving Others Your Cards

1. Click the export button shown above.

2. Press the Copy Cookie button. The code will be copied to your clipboard.

3. Give the code to others using whatever messaging app.

4. Others can now paste this code into the import box. And then click either "Import and Append" or "Import and Replace".
Import and Append - The cards that are originally here would not be deleted.
Import and Replace - The cards that are originally here would be deleted.

5. Others can now start a quiz with the cards that you just gave them.
